# 📊 Customer Churn Analysis

Business analysis project using the IBM Telco Customer Churn dataset to identify operational factors affecting customer retention and provide data-driven recommendations.

---

## 📌 Project Overview

This project analyzes customer churn in a telecommunications company using Microsoft Excel.

The analysis focuses on operational service quality, technical support availability, and network infrastructure performance to understand why customers decide to leave the service.

The project concludes with strategic recommendations that can help improve customer retention.

---

## 🎯 Objectives

- Clean and prepare customer data.
- Analyze customer churn behavior.
- Identify operational factors affecting churn.
- Build interactive dashboards using PivotTable and PivotChart.
- Provide business recommendations based on data.

---

## 🛠 Tools

- Microsoft Excel
- PivotTable
- PivotChart

---

## 📂 Dataset

IBM Telco Customer Churn Dataset

Source:
https://www.kaggle.com/datasets/blastchar/telco-customer-churn

Approximately 7,000+ customer records.

---

## 📈 Key Findings

### 1. Technical Support

Customers without Technical Support services have the highest probability of leaving the service.

### 2. Network Infrastructure

Fiber Optic customers experience the highest churn rate compared to other internet service types.

---

## 💡 Business Recommendations

- Conduct preventive maintenance on Fiber Optic infrastructure.
- Bundle free Technical Support for new Fiber Optic customers.
- Develop AI-based network anomaly detection for proactive maintenance.
